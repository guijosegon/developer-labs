# Security

Estudos de segurança aplicada: aplicações web, redes, criptografia e infraestrutura.

## Tópicos

- [ ] OWASP Top 10
- [ ] Autenticação e autorização: JWT, OAuth2, OIDC
- [ ] TLS/mTLS
- [ ] Secrets management (Vault, K8s secrets)
- [ ] SAST / DAST
- [ ] Supply chain security (SBOMs, assinatura de imagens)
- [ ] Threat modeling

## Estrutura

```
security/
```
