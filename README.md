# developer-labs

Repositório pessoal de estudos, experimentos e documentação técnica. O objetivo é consolidar conhecimento de forma prática — cada pasta representa uma área de aprendizado, com anotações, exemplos de código e projetos reais.

O conteúdo é construído de forma colaborativa: parte dos experimentos, estruturas e documentações são desenvolvidos com auxílio do **Claude (Anthropic)**, servindo também como exploração do uso de IA no fluxo de aprendizado técnico.

## O que você encontra aqui

| Pasta | Descrição |
|---|---|
| `docs/` | Conceitos teóricos, cheatsheets e decisões arquiteturais |
| `labs/` | Experimentos práticos organizados por tecnologia |
| `projects/` | Mini-projetos completos e integrações |
| `resources/` | Curadoria de links, livros e cursos |

## Colaboração com IA

Parte do conteúdo deste repositório é gerado ou assistido pelo **Claude (claude.ai)** — estruturas, exemplos, documentações e experimentos. Isso faz parte intencional do estudo: entender como usar IA como ferramenta no dia a dia de desenvolvimento.
